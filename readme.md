## Python FizzBuzz

Two function variations of "FizzBuzz".

### Usage

Clone this repo & run:

```sh
python example.py
```

Output:

```sh
1,2,Fizz,4,Buzz,Fizz,7,8,Fizz,Buzz,11,Fizz,13,14,FizzBuzz,16,17,Fizz,19,Buzz
1,2,Fizz,4,Buzz,Fizz,7,8,Fizz,Buzz,11,Fizz,13,14,FizzBuzz,16,17,Fizz,19,Buzz
```
